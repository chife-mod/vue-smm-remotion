# VUE — LinkedIn Carousel Wireframe Brief

## Project
VUE — NYC Building Compliance Platform

---

# Objective

Create a premium enterprise-style LinkedIn carousel series for VUE focused on NYC building compliance, portfolio visibility, and operational clarity.

The carousel should position VUE as a modern PropTech / compliance operating system for NYC building owners and property managers.

The visual direction should feel:
- Premium
- Cinematic
- Enterprise SaaS
- NYC-focused
- Data-driven
- Minimal and high-end

This is NOT an educational infographic carousel.

This IS a branded enterprise storytelling carousel.

---

# Primary Platform

- LinkedIn (primary)
- Facebook (secondary)
- Email usage possible

---

# Format

## Carousel Format
- Portrait Carousel
- 4:5 Ratio
- 1080 × 1350 px

---

# Creative Direction

## Core Theme
From chaos to clarity.

The carousel is about:
- Visibility
- Control
- Portfolio oversight
- Risk awareness
- Operational confidence

VUE should feel like:
- A command center
- A portfolio overview system
- A modern NYC building operations platform

- Cinematic NYC aerials
