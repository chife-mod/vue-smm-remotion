Five carousels · twenty-five slides · one mood

Working wireframe for the VUE LinkedIn carousel campaign. Left column = static slide layout, mood and copy. Right column = per-slide notes: Russian translation, visual proposal, brainstorm, Mobbin-searchable reference tags. Source brief: docs/brief.md.

1080 × 1350 · 4:5 portrait
5 posts · 5 slides each
Dark teal + neon-mint glow
Inter
Primary · LinkedIn

Creative direction

  • Premium · cinematic · enterprise SaaS
  • NYC-focused · data-driven · minimal
  • Dark mode · sapphire / electric blue accents
  • Glassmorphism panels · thin grid systems
  • Branded enterprise storytelling — not educational infographic
  • Core theme: From chaos to clarity
  • VUE feels like: command center · portfolio overview · ops platform

Anti-direction (no-go list)

infographic-carousel stock-cliche startup-flash neumorphism claymorphism cartoon-mascot crypto-bro 3d-blobs gradient-soup emoji-overload comic-fonts rewrite-client-copy

Sacred rules

  • Client copy is frozen — use exact text, no paraphrase
  • Slides carry hook typography only — full copy lives in caption
  • RU translation is mandatory per slide (idiomatic, not literal)
  • Every visual gets a placeholder until the design phase
